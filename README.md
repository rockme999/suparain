# SUPARAIN Website

SUPARAIN, Inc. 공식 웹사이트

## 기술 스택

- HTML5
- CSS3
- JavaScript (Vanilla)

## 로컬에서 실행하기

1. 저장소 클론:
```bash
git clone https://github.com/rockme999/suparain.git
cd suparain
```

2. 브라우저에서 `index.html` 파일 열기

또는 로컬 서버 실행:
```bash
# Python 3
python -m http.server 8000

# Node.js (http-server 설치 필요)
npx http-server
```

그 후 브라우저에서 `http://localhost:8000` 접속

## GitHub Pages 배포

이 저장소는 GitHub Pages를 통해 자동으로 배포됩니다.

- GitHub Pages URL: https://rockme999.github.io/suparain
- 커스텀 도메인: https://suparain.com (설정 필요)

## GitHub Pages 활성화 방법

1. 저장소 Settings → Pages 이동
2. Source 섹션에서:
   - Branch: `main` 선택
   - Folder: `/ (root)` 선택
3. Save 클릭
4. 몇 분 후 https://rockme999.github.io/suparain 에서 확인
